# TRA-202 — Collaborator awareness UI for calendar

**Linear:** [TRA-202](https://linear.app/travyl/issue/TRA-202/collaborator-awareness-ui-for-calendar)
**PR:** pending
**Status:** Planning
**Branch:** `feature/tra-202`

## Goal

Surface real-time collaborator state in the calendar UI — who's online, which view they're in, which day they're focused on, and which activity they have selected. No cursor tracking. Awareness only, similar to Google Docs "Sarah is on page 4."

## Scope

- Who's online on the trip (already broadcast via Supabase Realtime presence)
- Which view they're in: week or day
- Which day they're focused on
- Which activity they have selected (already rendered as avatar badges on EventBlock)

## Out of Scope

- Live cursor position tracking
- Scroll position sync
- Viewport size compensation

## Planned Implementation

- Extend `useCollaboratorPresence` to broadcast `viewMode` and `focusedDayIndex`
- Add presence indicators in `CalendarHeader` showing which view/day each collaborator is on
- Day column highlighting when collaborator is focused on a different day
- Tooltip on avatar showing "Sarah is viewing Week view, Day 3"
