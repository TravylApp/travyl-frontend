# TRA-204 — Trip sharing & post-it notes

**Linear:** [TRA-204](https://linear.app/travyl/issue/TRA-204/trip-sharing-and-post-it-notes)
**PR:** [#172](https://github.com/TravylApp/travyl-frontend/pull/172)
**Status:** Merged
**Branch:** `feature/tra-204`

## Goal

Google Docs-style trip sharing with private/link/public visibility, per-collaborator roles (viewer/commenter/editor), email invite flow, and free-floating post-it notes on the calendar canvas via Shift+Click.

## Completed (Database)

- `visibility` enum on `trips` (`private` | `link` | `public`)
- `link_permission` column on `trips` (permission level for link visitors)
- `trip_collaborators` table extended for pending invites (nullable `user_id` + `invite_email` + `invite_token` + `invite_expires_at`)
- `trip_notes` table with RLS for free-floating post-it notes

## Completed (Frontend)

- `ShareModal` — visibility selector, share link copy, link permission dropdown, collaborator invite/list
- `CollaboratorList` — avatar stack with role badges, pending invite badges, remove button
- `InviteBar` — email input + role selector + send button
- `LinkSharingSection` — visibility radio group + link permission dropdown + copy link button
- `/t/[token]` share route (Server Component) — validates token, renders calendar with read-only/edit mode based on `link_permission`
- `/invite/[token]` accept route — logged-in users accept invite, logged-out users redirect to signup then accept
- Next.js middleware protecting `/t/*`, `/invite/*`, `/trip/*`
- `TripPermissionContext` + `useEffectivePermission` hook — derives effective permission from ownership, collaborator record, or link permission

## Completed (Post-it Notes)

- `PostItNote` component — yellow sticky note with inline edit, dnd-kit drag
- Shift+Click on calendar canvas creates new note at click position
- Notes save on blur, delete on X button
- Notes visible to all collaborators with edit permission

## Deferred (Branch 2)

- Community fork/discover loop — public trips on home page
- Fork mechanic with attribution
- Trip discovery feed
