# TRA-192 — Calendar page redesign — full-page Outlook-style dashboard

**Linear:** [TRA-192](https://linear.app/travyl/issue/TRA-192/calendar-page-redesign-full-page-outlook-style-dashboard)
**PR:** [#140](https://github.com/TravylApp/travyl-frontend/pull/140)
**Status:** Merged
**Branch:** `feature/tra-192`

## Goal

Full-page calendar dashboard replacing the old tab-embedded trip view with an Outlook-inspired week/day grid, collapsible sidebar, slide-in detail panel, all-day row for flights/hotels, background images on event blocks, Google Docs-style header with collaborator avatars, click-to-create activities, and @dnd-kit drag-and-drop.

## Completed (Layout)

- Full-page `/trip/[id]` route with dedicated layout (no tabs)
- Collapsible left sidebar with trip summary + day list
- Main calendar area with week/day view toggle
- Slide-in `DetailPanel` from right when activity selected
- Google Docs-style header with trip title, date range, collaborator avatars

## Completed (Calendar Grid)

- `WeekView` — 7-day horizontal grid with hour slots
- `DayView` — single-day vertical grid with hour slots
- `DayColumn` — renders activities as positioned blocks
- `EventBlock` — activity card with background image, time, title, location
- All-day row at top for flights/hotels (spans full width)
- Click-to-create: click empty slot → creates activity at that time
- Drag-to-resize: drag bottom edge to adjust duration
- Drag-to-move: drag entire block to new day/time

## Completed (DnD)

- `@dnd-kit/core` for all drag interactions
- `useCalendarDnd` hook — handles move, resize, cross-day drag
- `DragOverlay` for visual feedback during drag
- Collision detection with `closestCenter`

## Completed (Theme)

- Travyl design system applied (Navy primary, Lustria headings, Satoshi body)
- Dark mode support via `next-themes`
- Background images on event blocks (Unsplash fallback)

## Known Issues / Deferred

- Mock flight/hotel data in all-day row (not yet connected to real data)
- Week view scroll sync with sidebar day list
