# TRA-203 — Add "For You" suggestion panel with SST recommendation engine

## Goal

Pinterest-style "For You" sidebar panel on the calendar dashboard that surfaces AI-powered activity suggestions and search results. Users drag suggestion cards directly onto the calendar to schedule them. Backend recommendation engine deployed on AWS via SST v3.

## Completed (Frontend)

- `ForYouPanel` — permanent right column with masonry grid, search box, category filter chips, loading/error/empty states
- `SuggestionCard` — full-image draggable cards with price/rating badges, hover overlay with "Drag to schedule"
- Drag-from-panel-to-calendar via `@dnd-kit/core` with `closestCenter` collision detection
- Panel swaps with `DetailPanel` when clicking an event on the calendar
- Dropped suggestions disappear from panel, restored on activity delete
- `useSuggestions` hook with mock data (10 Paris activities), search filtering, category filtering
- `useInteractionTracking` hook for fire-and-forget event logging

## Completed (SST Infrastructure)

- `sst.config.ts` + 4 infra definitions (`api.ts`, `storage.ts`, `events.ts`, `secrets.ts`)
- API Gateway v2 with CORS at `https://zp5ghw4drj.execute-api.us-east-1.amazonaws.com`
- 3 Lambda functions: `suggest` (with DynamoDB cache), `search` (stub), `interact` (EventBridge publish)
- DynamoDB table for recommendation cache with TTL
- S3 bucket for activity assets
- EventBridge bus for interaction events
- Supabase JWT auth on all endpoints

## Deferred (Future Tickets)

- Bedrock Titan embeddings for semantic search
